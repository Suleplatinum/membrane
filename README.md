## Simplified Membrane Model for Butane Isomers Separation
Zeolites are porous crystalline materials that have numerous uses, similar to the now popular metal organic frameworks (MOFs). They are very commonly used for the separation of gases 
based on the size of the molecule. If you have a gas molecule with kinetic diameter close to the pore size of the zeolite, you can achieve selective separation based on the size difference.
Gases like carbon dioxide and methan are routinely separated with various zeolites. The separation of hydrocarbons is another common application. For example the separation of linear alkanes from branched alkanes.


Here, I codify experimental data from Mitall et al's work developing a model to describe separation of n-butane and isobutane using MFI zeolite membrane.
The work uses a simple model to describe the permeance and diffusion of the hydrocarbon molecules in the zeolite pores.
